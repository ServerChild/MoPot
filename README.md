## 모팟(MoPot)

  * ##### 원하는 다양한 분야의 팟(모임)을 쉽게 검색할 수 있으며, 원하는 모임에 대한 구함과 신청을 쉽게 할 수 있도록 기능을 구현한 다양한 분야에 대한 팟(모임) 구하기 사이트입니다.
